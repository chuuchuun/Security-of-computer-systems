Signs a PDF file using the provided RSA private key bytes. The method adds a visible "Signed: ✅" annotation to the first page and embeds signature metadata in PDF info keywords.

Parameters
filePathThe path to the PDF file to be signed.
privateKeyBytesThe RSA private key in PKCS#1 or PKCS#8 format.
Returns
The full path of the newly saved signed PDF file.
Exceptions
ArgumentExceptionThrown when filePath is null or empty.
InvalidOperationExceptionThrown when the directory of the file path cannot be determined.
